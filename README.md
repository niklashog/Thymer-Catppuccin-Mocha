# Thymer Theme inspired by Catppuccin Mocha
Catppuccin Mocha Theme for Thymer notes application. The IDE for Notes, Tasks and Planning.

## Installation:
1. Open the .css file in this repository and copy (Ctrl + C) its contents.
2. Open up Thymer. Press Ctrl + P >  Upload Custom CSS > paste (Ctrl + V) > Click Save. 
3. Press Ctrl + P again > Set Theme & Dark or Light Appearance > Catppuccin Mocha.
